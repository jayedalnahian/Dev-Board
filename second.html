Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. Like document.getElementById(), document.getElementByClassNames(), document.getElementByTagName(), document.querySelector(), document.querySelectorAll(). Also we can use parentElement, childElement, previousElementSibling, nextElementSibling etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns all HTML content like tags and text contents. innerText returns visible text of a HTML element that is selected, it can not return hidden texts. textContent can return all text content inside a HTML element including hidden texts.

Question-3: What is event delegation in the DOM?

Event delegation is a way to handle events thru parent element, rather than attaching event listeners to individual child elements. Event delegation can make our code well decorated and easy to code.

Question-4: What is event bubbling in the DOM?

Event bubbling is a system of DOM, if an event triggered on a child element, it propagates up through its parent element. If an event occurs like click on an element, the event would also trigger on all of its parent elements. So the feature of moving through the higher elements is called Bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements I use document.createElement() and to add element to the DOM I use appendChild(), append() etc. While removing elements from the DOM use removeChild(), remove().
